# Portfolio Website

This is my personal **portfolio website** built with **React**, **Tailwind CSS**, and hosted on **Vercel**. It showcases my projects, skills, and includes a contact form to reach me.

## Features

- Fully **responsive layout** for desktop and mobile.
- **Projects showcase** with images and links.
- **Contact form** integrated with **EmailJS**.
- **Left-aligned labels** and **transparent card design**.
- Modern **font and colors** for a professional look.
- Smooth scrolling and hover effects for better UX.

## Tech Stack

- React
- Tailwind CSS
- EmailJS
- Lucide Icons
- Vercel (hosting)
